# UnifiedConfig

通过服务端根目录下的 serverconfig.yml 统一管理各个插件的配置文件。

## 功能

- 在服务端根目录创建 serverconfig.yml
- 通过修改 serverconfig.yml 统一管理其他插件的配置
- 支持热重载配置（无需重启服务器）
- 自动备份修改前的配置文件
- 自动检查更新
- 支持自定义消息
